A customized Discord Bot. Uses Python with discord.py library and a MariaDB database in the background.
Some features of this Bot include:
  - economy system, you can earn currency and spend it on items, gambling, pets, games and much more
  - moderation, it has several functions to moderate a discord server such as a unique ticket system, logging, suggestion forms and other small features to make a moderators life easier
  - prediction system, you can vote on League of Legends games and score points throughout the Esports season to win prizes. Voting automatically lock when games start
  - before the prediction system it implemented automatic locking of cells in google sheets, so users couldnt swap their votes - using the google sheet API
  - games, it includes games such as wordle or minesweeper
  - retrieving player stats from games such as League of Legends using the Riot Games API
